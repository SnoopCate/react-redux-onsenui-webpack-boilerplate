# react-onsenui-webpack-boilerplate

* React
* Redux
* CSS Modules
* PostCSS
* Webpack
* Cordova
* .editorconfig
* Ionicons

## Commands

```
# Web server
npm run start

# Dump the app in www folder
npm run dumpprod
npm run dumpdev

# Restore cordova platforms and plugin
npm run restore

# Run on device
npm run android
npm run iosEmulator
npm run ios
```
